## [1.0.0-alpha] - 2025-12-14

### Added
- Initial Alpha release
- SEC Form 4 insider trading data tracking
- Congressional trades monitoring (Beta - requires FINNHUB_API_KEY)
- AI-powered insights (Beta - requires GEMINI_API_KEY or OPENAI_API_KEY)
- Custom alert system with multiple notification channels
- Multi-channel notifications (email, push, Discord, Slack, SMS)
- Company and insider profile pages
- Market data integration
- Subscription billing via Stripe (Free, Plus, Pro, Enterprise tiers)
- User authentication and authorization
- Admin dashboard
- Support ticket system
- Federal Reserve calendar
- News aggregation
- Automated data scraping scheduler

### Security
- JWT-based authentication
- Password hashing with bcrypt
- Secure payment processing via Stripe
- Rate limiting and API security

### Documentation
- Comprehensive API documentation (Swagger UI)
- User guides and help documentation
- Release notes
