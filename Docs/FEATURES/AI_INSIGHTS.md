# AI Insights Feature Guide

Learn how to use TradeSignal's AI-powered insights to analyze insider trading patterns and get intelligent recommendations.

## Overview

AI Insights provides AI-powered analysis of insider trading data, including company analysis, trading signals, daily summaries, and an AI chat assistant. This feature is in **Beta** and requires API key configuration.

## Feature Status

⚠️ **Beta Feature:** AI Insights is currently in Beta status.

**Requirements:**
- `ENABLE_AI_INSIGHTS=true` in backend configuration
- Either `GEMINI_API_KEY` or `OPENAI_API_KEY` configured
- Backend service restarted after configuration

**API Options:**
- **Google Gemini:** Free tier (1,500 requests/day), recommended
- **OpenAI:** Paid, based on your API plan

**See:** [Release Notes](../RELEASE_NOTES_ALPHA_v1.0.0.md) for complete status

## Accessing AI Insights

### Navigation

1. Log in to TradeSignal
2. Click "AI Insights" in the main navigation
3. You'll see the AI Insights dashboard with multiple tabs

### Feature Availability

If AI Insights shows as "Not Available":
- Check backend configuration (ENABLE_AI_INSIGHTS, API keys)
- Verify API keys are valid and have quota
- Contact support for assistance

## AI Insights Features

### 1. Daily Summary

**What it does:**
- Provides AI-generated daily summary of top insider trading activity
- Highlights significant trades and patterns
- Shows top companies by trade value

**How to use:**
1. Navigate to AI Insights
2. Click "Daily Summary" tab (default)
3. Review AI-generated summary
4. Click on companies for details

**Update Frequency:**
- Auto-refreshes every 5 minutes
- Cached for performance
- Shows last update time

### 2. Trading Signals

**What it does:**
- AI-generated trading signals based on insider activity
- Identifies bullish/bearish patterns
- Provides signal strength and reasoning

**How to use:**
1. Click "Trading Signals" tab
2. Review signals for different companies
3. See signal type (Bullish/Bearish/Neutral)
4. Review strength and reasoning
5. Click "See More" for additional signals

**Signal Information:**
- **Ticker & Company:** Stock being analyzed
- **Signal Type:** Bullish, Bearish, or Neutral
- **Strength:** Strong, Moderate, or Weak
- **Buy Ratio:** Percentage of buys vs. sells
- **Total Value:** Combined value of trades
- **Reasoning:** AI explanation of signal

**Important:** These are AI-generated insights, not investment advice.

### 3. Company Analysis

**What it does:**
- AI-powered analysis of a specific company's insider trading
- Analyzes trading patterns over time
- Provides sentiment and key insights

**How to use:**
1. Click "Company Analysis" tab
2. Enter ticker symbol (e.g., NVDA)
3. Select days to analyze (7, 14, 30, 60, 90, 180, 365)
4. Click "Analyze"
5. Review AI analysis and insights

**Analysis Includes:**
- **Sentiment:** Bullish, Bearish, or Neutral
- **Analysis Text:** Detailed AI analysis
- **Key Insights:** Bullet points of important findings
- **Trade Statistics:** Number of trades, days analyzed

### 4. AI Chat Assistant

**What it does:**
- Interactive AI chat for asking questions about insider trading
- Answers questions about patterns, trends, specific companies
- Provides context and analysis

**How to use:**
1. Click "AI Chat" tab
2. Type your question in the input box
3. Press Enter or click Send
4. Review AI response
5. Ask follow-up questions

**Example Questions:**
- "Which companies have the most insider buying this month?"
- "What are the biggest trades this week?"
- "Explain NVDA's recent insider activity"
- "Show me recent CEO purchases over $1M"

**Tips:**
- Be specific in your questions
- Ask about specific tickers, dates, or patterns
- Use follow-up questions for clarification

## Understanding AI Responses

### Response Time

- **Typical:** 2-10 seconds
- **Factors:** Question complexity, API load, cache status
- **Caching:** Responses cached for 24 hours

### Response Quality

- **AI-Generated:** Responses are generated by AI models
- **May Contain Errors:** AI can make mistakes or provide incomplete information
- **Use for Context:** Treat as informational, not definitive
- **Verify Important Info:** Cross-check critical information

### Limitations

- **Beta Status:** Feature is still improving
- **Data Dependency:** Quality depends on available data
- **API Limits:** Subject to API rate limits and quotas
- **Not Real-Time:** May use cached or slightly outdated data

## Troubleshooting

### AI Insights Not Available

**Solutions:**
1. Check backend has `ENABLE_AI_INSIGHTS=true`
2. Verify API key is configured (GEMINI_API_KEY or OPENAI_API_KEY)
3. Restart backend after configuration
4. Check API key is valid and has quota
5. Contact support for assistance

### Slow Responses

**Expected:**
- AI processing takes 2-10 seconds
- This is normal behavior

**If Unusually Slow:**
1. Check API status (Gemini/OpenAI)
2. Verify API quota/rate limits
3. Wait and retry
4. Contact support if persistent

### Inaccurate Responses

**Note:** AI can make mistakes. This is expected in Beta.

**Solutions:**
1. Verify input (ticker, dates) is correct
2. Try rephrasing question
3. Be more specific
4. Report issues to support

### Error Messages

**Common Errors:**
- "AI service unavailable" - Check API key and backend config
- "Rate limit exceeded" - Wait before retrying
- "Invalid request" - Check input format

**Solutions:**
1. Read error message for details
2. Check [Troubleshooting Guide](../TROUBLESHOOTING.md)
3. Contact support if needed

## FAQ

**Q: Do I need to configure AI Insights?**  
A: The backend administrator needs to configure API keys. End users don't need to do anything if it's already set up.

**Q: Which AI provider is better?**  
A: Both Gemini and OpenAI work well. Gemini free tier is good for getting started. OpenAI may have better performance for complex queries.

**Q: Are AI insights accurate?**  
A: AI insights are generated by AI models and may contain errors. Use for informational purposes and verify important information.

**Q: Can I trust trading signals?**  
A: Trading signals are AI-generated and should not be considered investment advice. Always do your own research.

**Q: How often are insights updated?**  
A: Daily Summary auto-refreshes every 5 minutes. Other features update when you request them. Responses are cached for 24 hours.

**Q: Is my data sent to AI providers?**  
A: Only public trading data is sent with queries. No personal information is included. See Privacy Policy for details.
